Bootstrap Customize

sass 설정하기

node js 설치 필요
npm init 실행; project root 에 package.json파일이 생성됨
npm install bootstrap
/node_modules/bootstrap/scss에 가서 _valiables.scss에 보면 기본 설정된 값이 있음
이걸 수정하기 위해서는 /scss/main.scss 를 생성
/sass/main.scss 파일 생성, 기본 color 값 변경 
// main.scss
  $primary: #c29938; // 이렇게 하면 기본 값을 override 함.
  //bootstrap.scss 파일에 링크가 있어서 전체를 가져옮, scss 확장자 불필요
  @import '../node_modules/bootstrap/scss/booststrap';

sass 컴파일하기

vs code extenstion에서 live sass compiler 설치, 자동으로 scss를 컴파일해서 css로 변환
vs code settings : live sass format, 컴파일된 파일의 경로, 파일 위치 등을 정의
live sass formats > settings: formats > edit in settings.json
"/css/main.min-5-3.css"에 main.scss에서 정의한 내용으로 bootstrap 설정을 덮어 씀 
"liveSassCompile.settings.formats": [
        // {
        //     "format": "expanded",
        //     "extensionName": ".css",
        //     "savePath": null,
        //     "savePathReplacementPairs": null
        // }
        {
          "format": "compressed",
          "extensionName": ".min-5-3.css",
          "savePath": "/css",
          "savePathReplacementPairs": null
        }
    ],
vs code 하단 작업표시줄에서 watch sass 눌러야함
/css/main.min-5-3.css 파일이 생성됨 -> 이파일을 css로 설정함
theme color에 사용자 정의 색 추가하기 
// import bootstrap
// 이렇게 하면 기본 값을 override 함.
$primary: #c29938; 
$light: #f5e9cf;

// theme color에 내가 설정한 색을 추가하기
@import "../node_modules/bootstrap/scss/functions";
@import '../node_modules/bootstrap/scss/variables';

$custom-theme-colors: (
  "altlight": rgba(241, 183, 237, 0.733),
  "altdark": rgb(100, 3, 3)
);

$theme-colors: map-merge($custom-theme-colors, $theme-colors);

@import '../node_modules/bootstrap/scss/bootstrap';
